
 # 简介

 ## 安装

 参考 [官方文档](https://elixir-lang.org/install.html)

 验证安装 ：
 
 ```elixir
 elixir --version
 或者
 elixir -v
 ```

输出

```elixir
╰─± elixir -v
Erlang/OTP 27 [erts-15.0.1] [source] [64-bit] [smp:10:10] [ds:10:10:10] [async-threads:1] [jit]

Elixir 1.17.2 (compiled with Erlang/OTP 27)
```

## 交互模式

Elixir安装会会有3个新的命令行工具。
- `iex`:  交互式工具
- `elixir`: 脚本执行工具
- `elixirc`: 编译工具
